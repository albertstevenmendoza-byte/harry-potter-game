The Game
🧙
5 Iconic Characters

Harry, Hermione, Ron, Voldemort, or Snape — each with unique HP totals and spell-power multipliers.

🪄
5 Legendary Wands

Holly, Vine, Elder, Walnut, and Ash — each grants a damage bonus to its affinity spell.

📜
20 Canonical Spells

Type the real spell name or lose your turn. Expelliarmus to Avada Kedavra, all included.

⚖️
Diminishing Returns

Cast the same spell twice in a row and it deals only 50 % damage. Variety wins duels.

🏆
Best-of-Five

First player to claim three rounds wins the match. Every round is a fresh duel.

✨
Rich Terminal UI

Animated HP bars, coloured spell trajectories, character panels, and cinematic round transitions.

Every spell is validated against this hardcoded registry. Type it wrong — lose your turn.

Get Started

Installation

Three commands. Then duel.

1
Clone the repository

Download the project to your machine.

git clone https://github.com/YOUR_USERNAME/hp-duel.git cd hp-duel
2
Install the one dependency

Requires Python 3.10+ and the Rich library for the terminal UI.

pip install -r requirements.txt
3
Start the duel

Both players sit at the same keyboard. The game handles the rest.

python hp_duel.py
4
Remember the rules

Type a spell name on your turn. Type LIST for a free spell reference. First to win 3 rounds wins the match. Casting the same spell twice in a row deals only 50% damage — mix it up.
